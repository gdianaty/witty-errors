Witty Errors Asset Pipeline

For the Node.JS or Lightweight version of Witty, basically just holds assets on an external server. Probably at wittypl.gmdianaty.com if you need the assets.

The assets list is as follows:

wittypl.gmdianaty.com
├── assets
│   ├── 401.gif
│   ├── 403.gif
│   ├── 404.gif
│   ├── 500.gif
│   └── 503.gif
├── bin
│   ├── theme.css
│   └── theme.scss

So if you want the GIF for Error 403, you just request wittypl.gmdianaty.com/assets/403.gif to your HTML
